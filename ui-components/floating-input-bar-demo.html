📝 Floating Input Bar Demo

Barra de texto flotante ancho completo (20mm) con botón 10x10mm

✨ Características

  • 📏 Dimensiones: 20mm de alto, ancho completo, botón 10x10mm
  • 📍 Posición: Flotante en bottom, draggable
  • ⌨️ Controles: Enter para enviar, ESC para limpiar
  • 🎤 Voice: Integración con reconocimiento de voz
  • 💾 Persistencia: Recuerda la posición
  • 👁️ Mira abajo: La barra está en la parte inferior de la pantalla

📜 Scroll Test

Este contenido extra permite probar el comportamiento de la barra al hacer scroll. La barra flotante permanece visible en bottom mientras navegas el contenido.

Si activas el modo auto-hide, la barra se ocultará al hacer scroll down y reaparecerá al hacer scroll up, dándote más espacio visual.

Características del drag: Puedes arrastrar la barra desde el handle superior (línea horizontal) para reposicionarla verticalmente. La posición se guardará automáticamente en localStorage.
