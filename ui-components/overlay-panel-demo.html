📱 Overlay Panel Demo

¿Qué es un Overlay Panel?

Panel superpuesto semi-transparente que te permite ver detalles sin perder de vista la pantalla anterior.

Ventajas:

  • Fondo semi-transparente con blur: ves la pantalla debajo
  • 3 tamaños: pequeño (320px), mediano (480px), grande (640px)
  • Scroll interno: contenido largo sin problema
  • Cierra con: botón ×, click fuera, tecla ESC

🎮 Emoji Buttons con Overlays

Click simple en cada botón para ver detalles en overlay

📏 Tamaños de Panel

Pequeño (320px)

Ideal para confirmaciones, detalles rápidos, alertas

Mediano (480px)

Ideal para detalles de agentes, plugins, configuración

Grande (640px)

Ideal para editar prompts, código, contenido extenso
