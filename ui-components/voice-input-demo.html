🎤 Voice Input Demo

Reconocimiento de voz (Speech-to-Text) con Web Speech API

🚀 Cómo Usar

  • Presiona el botón 🎤 para activar el micrófono
  • Habla claramente hacia tu micrófono
  • El texto aparecerá en tiempo real mientras hablas
  • Presiona 🔴 para detener la grabación
  • Usa 🌐 para cambiar el idioma
  • Usa 🗑️ para limpiar el texto
  • También puedes escribir manualmente en el campo

1️⃣ Input Simple

Input de una línea con reconocimiento de voz. Ideal para búsquedas, formularios cortos, comandos por voz.
