📱 FloatingActionBar - Demo

🎯 Barra flotante minimalista para móvil

✅ Swipe down: Ocultar barra (arrastra el handle)
✅ Scroll: Auto-hide activado (scroll down = oculta, scroll up = muestra)
✅ Glassmorphism: Fondo con blur
✅ TouchActionButtons: Botones multi-acción integrados

🎮 Controles

✨ Características

Gestos Móvil

  • Swipe down en el handle para ocultar
  • Auto-hide cuando haces scroll down
  • Auto-show cuando haces scroll up
  • Vibración háptica en interacciones

Visual

  • Glassmorphism con backdrop blur
  • Bordes redondeados superiores
  • Shadow suave
  • Animaciones fluidas
  • Safe-area support (iOS notch)

Botones

  • Scroll horizontal si hay muchos
  • Integración con TouchActionButton
  • Spacing configurable
  • Add/remove dinámico
